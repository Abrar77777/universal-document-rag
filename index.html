Portfolio Project

Universal Document RAG Assistant

A multi-format Retrieval-Augmented Generation system that ingests PDFs, Word documents, text files, CSVs, and Excel workbooks, retrieves the best evidence chunks, answers with citations, and profiles numeric data with visual analysis.

FastAPI Backend Streamlit Frontend FAISS Retrieval Groq LLM CSV/Excel Analytics

Universal Ingestion

Loads and chunks PDF, DOCX, TXT, Markdown, JSON, CSV, XLS, and XLSX files.

Grounded Answers

Retrieves top-k chunks, sends context to Groq, and returns citations and confidence.

Numeric Insight

Detects tabular data, summarizes numeric columns, and prepares chart-ready payloads.

Architecture

1. UploadUsers add documents through the Streamlit UI.
2. ChunkText is cleaned and split with overlap for recall.
3. EmbedChunks are embedded and stored in FAISS.
4. RetrieveTop-k chunks are selected for each question.
5. AnswerGroq generates a grounded answer with sources.

Run The Demo

Clone the repository, add your API key to .env, then run the Windows launcher:

.\run_app.ps1

If PowerShell blocks scripts, run:

powershell -ExecutionPolicy Bypass -File .\run_app.ps1

Then open http://127.0.0.1:8501.
